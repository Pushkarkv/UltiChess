# Ultimate Chess Platform

A premium, feature-rich chess platform with Stockfish engine integration, AI battle arena, training modes, and custom piece variants.

## Features (Phase 1)
- ♟ Standard chess with full rule enforcement
- 🎨 Premium dark UI with glassmorphism design
- 🖱️ Drag-and-drop & click-to-move
- 📊 Stockfish engine analysis with eval bar
- 📋 PGN import/export, FEN position setup
- ⏱️ Chess clocks with multiple time controls
- 🔄 Move history with navigation
- 🔊 Sound effects for moves
- 🌗 Dark/Light theme toggle

## Coming Soon
- 🧩 Fairy Stockfish custom pieces & variable boards
- ⚔️ AI vs AI battle arena
- 🧠 Visualization & calculation training
- 🏆 Puzzle Storm mode

## Tech Stack
- Vite 5 + Vanilla JavaScript
- chess.js for game logic
- Custom SVG board renderer
- Web Audio API for sounds

## Getting Started
```bash
npm install
npm run dev
```
